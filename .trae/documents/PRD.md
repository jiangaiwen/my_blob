# 个人博客网站 PRD 文档

## 1. 产品概述

一款对标 poetize.cn 风格的纯静态个人博客网站，采用 Vue3 + Vite + Naive UI + Tailwind CSS 技术栈构建。网站定位为简约文艺风的个人文学作品展示平台，支持亮色/深色主题切换、全屏必应壁纸背景、滚动入场动画，旨在为访客提供沉浸式的阅读体验。

## 2. 核心功能

### 2.1 功能模块

1. **首页**

   * 全屏必应随机壁纸背景（每次刷新自动切换）

   * 左侧个人信息侧边卡片

   * 右侧文章流式卡片列表

   * 顶部导航菜单栏

   * 站点统计展示

   * 最新文章、热门文章模块

   * 所有模块自带滚动入场动画

2. **文章列表页**

   * 文章分类筛选

   * 标签筛选

   * 分页布局

   * 卡片式展示文章基础信息

   * 列表条目依次渐入展示

3. **文章详情页**

   * 完整文章标题、发布时间、分类标签

   * Markdown 正文渲染（代码高亮）

   * 模拟点赞、评论模块UI

   * 段落图片分步动画呈现

4. **时间线页面**

   * 按时间倒序排版

   * 时间轴样式展示过往动态

   * 时间节点左右交替弹出动画

5. **相册页面**

   * 瀑布流图片墙布局

   * 图片 hover 放大预览

   * 分类展示图集

   * 图片滚动缩放入场

6. **留言板页面**

   * 留言列表静态展示

   * 留言输入交互UI

   * 留言气泡逐个渐入

7. **友情链接页**

   * 友链卡片网格布局

   * 站点跳转链接

   * 卡片批量错落出场动画

8. **关于本站页**

   * 个人简介

   * 站点介绍

   * 站点数据展示

   * 文本模块平滑显现

### 2.2 通用交互功能

| 功能   | 描述                       |
| ---- | ------------------------ |
| 壁纸功能 | 调用第三方必应壁纸接口，解析图片地址自动切换背景 |
| 主题切换 | 顶部明暗模式切换按钮，切换样式并本地缓存用户偏好 |
| 导航交互 | 导航菜单点击跳转，移动端自动折叠为侧边抽屉菜单  |
| 文章交互 | 文章卡片悬浮动效，点击跳转详情          |
| 相册交互 | 图片懒加载，点击弹出大图预览弹窗         |
| 基础交互 | 平滑滚动、返回顶部按钮、面包屑导航        |
| 动画交互 | 滚动触发元素动画，路由切换重置动画状态      |

## 3. 技术栈

| 技术    | 版本/说明                                     |
| ----- | ----------------------------------------- |
| 前端框架  | Vue3 + Vite                               |
| 开发语言  | 原生 JavaScript（不使用 TypeScript）             |
| UI组件库 | Naive UI                                  |
| 路由管理  | Vue Router 4                              |
| 状态管理  | Pinia                                     |
| 样式方案  | Tailwind CSS + Scoped样式 + animate.min.css |
| 动画库   | wow\.min.js + Animate.css                 |
| 网络请求  | Axios（仅用于必应壁纸接口）                          |
| 文本渲染  | Markdown解析器（marked + highlight.js）        |

## 4. 视觉设计规范

### 4.1 设计风格

* 简约文艺风，参考 poetize.cn

* 毛玻璃效果、圆角设计

* 渐变阴影样式

* 支持亮色/深色主题

### 4.2 配色方案

**亮色主题**

| 用途   | 颜色                    |
| ---- | --------------------- |
| 主色   | #6366f1 (靛蓝)          |
| 背景   | #f8fafc               |
| 卡片背景 | rgba(255,255,255,0.8) |
| 文字主色 | #1e293b               |
| 文字次色 | #64748b               |

**深色主题**

| 用途   | 颜色                 |
| ---- | ------------------ |
| 主色   | #818cf8 (浅靛蓝)      |
| 背景   | #0f172a            |
| 卡片背景 | rgba(30,41,59,0.8) |
| 文字主色 | #f1f5f9            |
| 文字次色 | #94a3b8            |

### 4.3 字体规范

* 标题字体：思源宋体 / Noto Serif SC

* 正文字体：思源黑体 / Noto Sans SC

* 代码字体：JetBrains Mono / Fira Code

### 4.4 动画规范

* 全局引入 wow\.js 和 animate.css

* 初始化 WOW 实例，默认触发滚动动画

* 页面元素添加 wow 基础类名

* 搭配 fadeIn、fadeInUp、fadeInLeft、zoomIn 等动画样式

* 自定义动画延迟、动画时长，错开元素出场顺序

## 5. 响应式断点

| 设备  | 断点             |
| --- | -------------- |
| 桌面端 | ≥1024px        |
| 平板端 | 768px - 1023px |
| 移动端 | <768px         |
